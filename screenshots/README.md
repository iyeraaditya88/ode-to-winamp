# Screenshots

Drop image/GIF assets here and they'll show up in the main README's
**Screenshots** section. Suggested files (rename or add as you like):

| File | Shows |
|------|-------|
| `grid.png` | The WebGL liked-songs sphere (the hero shot) |
| `now-playing.png` | Now Playing with synced lyrics + equalizer |
| `music-taste.png` | The Music Taste genre radar |
| `demo.gif` | A short screen recording (best for adoption!) |

Tips:
- A 5–10s GIF of dragging/zooming the grid and playing a track is the single
  most effective thing you can add.
- Keep images < ~1–2 MB so the README loads fast (compress PNGs / use an
  optimized GIF or a short `.mp4`/`.webm` linked from the README).
